Overview

This executable test suite is based on the following OGC specifications. Please find WCS 2.0 ATS in the corresponding documents.

  • OpenGIS® Web Coverage Service(WCS) 2.0 Interface Standard - Core (OGC 09-110r3)
  • OpenGIS® Web Coverage Service 2.0 Interface Standard - KVP Protocol Binding Extension (1.0) (OGC 09-147r1)
  • OpenGIS® Web Coverage Service 2.0 Interface Standard - XML/POST Protocol Binding Extension (1.0) (OGC 09-148r1)
  • OpenGIS® Web Coverage Service 2.0 Interface Standard - XML/SOAP Protocol Binding Extension (1.0) (OGC 09-149r1)
