Overview

This test suite is based on the following OGC specifications:

  • OWS Context Conceptual Model, version 1.0.0 [OGC 12-080]
  • OWS Context ATOM Encoding, version 1.0.0 [OGC 12-084]
  • Geography Markup Language (GML) -- Simple features profile, version 1.0.0 [OGC 06-049r1]

Not all of GML 3.1.1 is covered by this test suite. The test data make use of only those elements allowed by the GEORSS simple profile.
